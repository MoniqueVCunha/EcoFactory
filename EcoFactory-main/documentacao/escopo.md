# Escopo do Projeto — EcoFactory

## Problema
A EcoFactory controla máquinas, produção, consumo de energia/água e ocorrências
de segurança em planilhas soltas, dificultando o controle e a tomada de decisão.

## Objetivo
Aplicação web para cadastrar, consultar, atualizar, excluir e analisar esses dados.

## Público
Gestores e operadores da fábrica.

## Funcionalidades obrigatórias
- CRUD completo de máquinas
- Cadastro/consulta de produção
- Dashboard com indicadores
- Persistência em PostgreSQL
- Integração Front-End ↔ API
- Validação dos formulários principais
- README com instruções

## Funcionalidades bônus (se sobrar tempo)
- Sustentabilidade
- Ocorrências de segurança
- Filtros/gráficos

## Tecnologias
React + Vite (front), Node.js + Express (back), PostgreSQL (banco), Git/GitHub, git init
